Architecture & Design

Built to showcase a "Black & Gold" premium aesthetic, this portfolio is engineered for speed and accessibility. I chose to avoid heavy frameworks like Bootstrap to ensure the code remains lean and easy to maintain.

Technical Features

  • Intersection Observer: Custom JS to trigger animations only when elements enter the viewport.
  • Flex & Grid: A hybrid layout system that adapts to everything from 4K monitors to mobile devices.
  • Formspree Integration: A serverless contact system for instant user communication.
